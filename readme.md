# Olaf Cli
Olaf(奥拉夫)：项目初始化工具，接下来会加入React Native(expo)项目与触屏版项目的支持，敬请期待:blush:

## 特性
### **一条命令初始化React + Next + Redux + Koa应用**

## 使用

### 安装
```bash
> npm install -g olaf-cli
```
### 查看帮助
```bash
> olaf -h
```

### 初始化项目
```bash
> olaf i
```
